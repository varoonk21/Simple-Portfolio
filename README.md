# Simple Portfolio

A modern, responsive personal portfolio website built with React, Vite, and Tailwind CSS.

## Overview

This is a clean and minimalist portfolio template designed to showcase your skills, projects, and experience. It features a sleek dark mode support, smooth animations, and a mobile-first responsive design.

## Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/varoonk21/Simple-Portfolio.git
cd Simple-Portfolio
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

Create an optimized production build:
```bash
npm run build
```

### Preview

Preview the production build locally:
```bash
npm run preview
```

### Lint

Check code quality with ESLint:
```bash
npm run lint
```

## Customization

To personalize this portfolio:

1. **Update Title** - Edit `<title>` in `index.html`
2. **Modify Content** - Update text, images, and links in `App.jsx`
3. **Add Resume** - Place your resume as `resume.pdf` in the `public/` folder
4. **Change Colors** - Customize Tailwind colors in `App.css` or directly in components
5. **Update Social Links** - Modify social media URLs in the contact section
